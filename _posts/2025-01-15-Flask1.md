# WSL 설치 전 필수 설정: "Linux용 Windows 하위 시스템" 활성화하기


## 1. WSL2 설치 및 Ubuntu 22.04 LTS 설정

### WSL2 설치
1. **WSL 설치**
```
wsl --install
# 설치 후 재부팅
# username, password 설정
```
2. **WSL 버전 확인**
```
wsl -l -v
```
3. **WSL 기본 버전을 WSL2로 설정**
```
wsl --set-default-version 2
```
4. **WSL 업데이트**
```
wsl --update
```
5. **사용 가능한 배포판 확인**
```
wsl --list online
```
